## 已完成

### 简历预处理

- 格式检测；
- 只修格式的清洗；
- 首次清洗前原稿留存；
- STAR 拆解；

- `resume-audit.json` 与 `resume-audit.md`。

### 五维分析

- 硬技能；
- 经验深度；
- 业务方向；
- 软性匹配；
- 岗位质量；
- 五维等权总分；
- 岗位质量扣分信号；
- `resume_hash + job_hash + scoring_version` 缓存；
- Top 5 深度分析；
- 未联网薪酬状态。

### 定制可信链

- 定制简历；
- 200 字开场白 Validator；
- 占位符检查；
- Diff 驱动七类 Changelog；
- 不可变身份和履历检查；
- 输入快照。

### 架构图

- 流水线内证据图；
- 独立项目描述充分性检查；
- 最多三条聚焦追问；
- 根据用户有序步骤生成 Mermaid；
- 信息不足时不补写节点。

### 报告与 ATS

- Run 阶段和进度；
- Resume Audit 摘要；
- 五维评分；
- 岗位质量；
- Cover Letter 条件标签；
- 每岗位 ATS 单栏 `resume.html`；
- ATS 禁止元素和隐藏文字检查；
- PDF 渲染器未执行时明确报告。

### Skill 包

- 六个 Skill 的声明与实现边界同步；
- 移除失效脚本引用；
- 六个 `agents/openai.yaml`；
- 主 Skill 指向 v0.3 需求基线。
